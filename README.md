# libftTester (2022+)
libft-00のtesterです
[Tripouille libftTester](https://github.com/Tripouille/libftTester)のリポジトリを参考にしてます

## Commands
make m = launch mandatory tests  

make vs [function name] = open the corresponding tests in vscode ex: `make vscalloc`
